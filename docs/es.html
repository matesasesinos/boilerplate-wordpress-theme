El proposito de este documento es describir el theme y su correcto uso. Este theme esta abierto a propuestas pero la intención es que se trabaje en todo proyecto futuro con la misma estructuras y misma convención de código.

Descripción del theme

El theme sigue el patrón psr-4 utilizando Composer.
También se utiliza Webpack y Babel para compilar y minificar JS y CSS.
Para crear opciones, metaboxes y bloques gutenberg simples, se utiliza Carbon Fields.
